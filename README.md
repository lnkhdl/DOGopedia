## DOGopedia

A simple Laravel project which uses a public [TheDogAPI](https://thedogapi.com/) to load data and images of dogs.

Different tools and libraries were used:
* Tailwind CSS,
* Alpine.js,
* Livewire,
* Lightbox2,
* Spatie Laravel View Models.
